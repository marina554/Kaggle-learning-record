#Beginner Titanic Kaggle Model/ Titanic Kaggle 初級モデル 

## English
This notebook is a beginner-level learning model using the Titanic dataset from Kaggle.  
Features used: `Sex`, `Pclass`, and `Age`. A RandomForest model is used for prediction.  
Learning goals:
- Get familiar with Kaggle workflow
- Understand basic data preprocessing
- Practice Python and machine learning fundamentals

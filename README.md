# Titanic Kaggle 初級モデル / Beginner Titanic Kaggle Model

## 日本語
このノートブックは、Kaggle の Titanic データセットを使用した初級者向け学習用モデルです。  
特徴量は `Sex`、`Pclass`、`Age` のみを使用し、RandomForest で生存予測を行います。

### 使用ライブラリ
- pandas
- numpy
- scikit-learn

### 学習目的
- Kaggle の使い方に慣れる
- データ前処理（欠損値補完、特徴量選択）の基礎を理解
- Python と機械学習の基礎を実践

### Kaggle スコア
- 初期提出: 約 0.765
- 改良後: 約 0.787 （ランダム性による変動あり）

### 注意点
- 初心者向けのモデルであり、性能改善の余地があります
- データの可視化や追加特徴量でスコア向上が可能です

This notebook is a beginner-level learning model using the Titanic dataset from Kaggle.
